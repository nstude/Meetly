sequenceDiagram
    Клиент->>Сервер: Запрос (HTTP POST): Страница авторизация (авторизация/регистрация)
    activate Сервер
    Сервер->>База Данных: Запрос (SQL): Отправка данных (логин и пароль)
    activate База Данных
    База Данных-->>Сервер: Ответ (Данные): Ответ (Если ошибка - код ошибки)
    deactivate База Данных
    Сервер-->>Клиент: Ответ (HTTP): HTML главной страницы (в случае корректного входа/авторизации)
    deactivate Сервер

    Клиент->>Клиент: Отображение страницы пользователя

    Клиент->>Сервер: Запрос (HTTP POST): Создание нового поста (со страницы пользователя)
    activate Сервер
    Сервер->>База Данных: Запрос (SQL): Добавление нового поста в БД
    activate База Данных
    База Данных-->>Сервер: Ответ (Подтверждение): Успешное добавление поста
    deactivate База Данных
    Сервер-->>Клиент: Ответ (HTTP): Подтверждение создания поста (остаёмся на главной)
    deactivate Сервер

    Клиент->>Клиент: Отображение страницы пользователя

    Клиент->>Сервер: Запрос (HTTP POST): Создание комента к посту (со страницы пользователя)
    activate Сервер
    Сервер->>База Данных: Запрос (SQL): Добавление нового комента в БД
    activate База Данных
    База Данных-->>Сервер: Ответ (Подтверждение): Успешное добавление комента к посту
    deactivate База Данных
    Сервер-->>Клиент: Ответ (HTTP): Подтверждение создания комента (остаёмся на главной)
    deactivate Сервер

    Клиент->>Клиент: Отображение страницы пользователя

    Клиент->>Сервер: Запрос (HTTP POST): Переход на страницу групп пользователя
    activate Сервер
    Сервер->>База Данных: Запрос (SQL): Получение данных о всех группах
    activate База Данных
    База Данных-->>Сервер: Ответ (Данные): Информация о группах
    deactivate База Данных
    Сервер-->>Клиент: Ответ (HTTP): Новая HTML страница со списком всех групп (в случае корректного ответа)
    deactivate Сервер
